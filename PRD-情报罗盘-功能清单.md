# 情报罗盘模块 — 功能清单

| 模块 | 功能点 | 说明 | 优先级 |
|------|--------|------|--------|
| **文章详情页-文章展示区** | 在文章中标记实体 | 在文章中标记实体 | P0 |
| | 文章来源调整 | 文章来源位置及交互调整 | P0 |
| **文章详情页-实体展示区** | 实体面板展示 | 在页面右侧展示文章中出现的所有实体 | P0 |
| | 实体分组展示 | 实体按类型分组展示 | P0 |
| | 实体卡片联动 | 悬停文章实体时面板对应卡片联动高亮 | P1 |
| **实体详情页** | 实体详情展示 | 通过右侧抽屉展示实体的完整信息 | P0 |
| | 战技指标展示 | 以卡片网格形式展示核心战技指标 | P0 |
| | 技术规格展示 | 以表格形式展示技术参数 | P0 |
| | 挂载装备展示 | 展示可挂载武器装备 | P0 |
| | 使用方信息展示 | 展示主要使用方信息 | P0 |
| | 动态追踪展示 | 以时间轴形式展示关键事件动态 | P0 |
| | 实体内容切换 | 抽屉内切换实体时直接替换内容 | P0 |
