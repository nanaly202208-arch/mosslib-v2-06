# MossLib 情报罗盘模块 PRD

## 1. 产品概述

### 1.1 产品定位
情报罗盘是 MossLib 平台的核心情报分析组件，通过多维度数据可视化与智能关联分析，帮助情报分析师快速掌握目标实体的全方位信息，发现隐藏关联，支撑决策制定。

### 1.2 目标用户
- 情报分析师
- 军事研究人员
- 安全领域专家
- 战略规划人员

### 1.3 核心价值
- **全景视图**：一站式展示实体核心情报要素
- **关联洞察**：智能挖掘实体间隐含关系
- **动态追踪**：实时监测目标实体最新动态
- **高效交互**：简洁直观的操作体验

## 3. 功能详细设计

### 3.1 实体概览区

#### 3.1.1 功能描述
展示目标实体的核心身份信息，包括名称、别名、承包商、状态标签等关键元数据。

#### 3.1.2 信息展示
| 字段 | 说明 | 示例 |
|------|------|------|
| 实体原文名 | 原始语言名称 | Orion |
| 实体中文名 | 中文译名及别名 | 猎户座 / Inokhodets-RU |
| 承包商 | 研制/生产厂商 | Kronshtadt 公司（原 Transas）|
| 别名 | 其他常用称呼 | Inokhodets-RU |
| 状态标签 | 现役/研发/退役等 | 现役、原产国 · Russia |
| 关键节点 | IOC、首飞、总产量 | IOC: 2020-12、首飞: 2016-05 |

#### 3.1.3 交互设计
- **标签筛选**：点击标签可快速筛选同类型实体
- **承包商跳转**：点击承包商名称跳转至厂商详情页
- **别名提示**：hover 显示别名使用场景说明

#### 3.1.4 视觉规范
- 实体名称：28px，font-weight: 800，颜色 #1E293B
- 中文/别名：16px，font-weight: 400，颜色 #64748B
- 标签区域：位于标题上方，margin-bottom: 10px
- 元数据区域：13px，颜色 #64748B，带图标前缀
- 图标颜色：蓝灰色 #64748B

---

### 3.2 战技指标区

#### 3.2.1 功能描述
以卡片形式展示实体的核心战术技术性能指标，支持快速对比与深度分析。

#### 3.2.2 KPI 卡片设计
| 指标项 | 数值 | 单位 | 补充说明 |
|--------|------|------|----------|
| 最大续航 | 24 | hr | 8,000m 升限持续飞行 |
| 作战半径 | 300 | km | 最大航程同步 |
| 最大起飞重量 | 1,150 | kg | 有效载荷 250kg |
| 系统编制 | 3 | 架/套 | 俄国防部订购7套(21架) |

#### 3.2.3 技术规格表
| 规格项 | 数值 | 规格项 | 数值 |
|--------|------|--------|------|
| 翼展 | 16.3 m | 巡航速度 | 200 km/h |
| 长度 | 8 m | 升限 | 7,500 m |
| 高度 | 3.2 m | 最大航程 | 300 km |
| 有效载荷 | 250 kg | 滑翔炸弹射程 | 30 km |

#### 3.2.4 交互设计
- **卡片悬停**：边框高亮 + 轻微上浮阴影
- **数值对比**：支持多实体指标对比视图
- **规格筛选**：按类别筛选技术参数
- **导出功能**：支持导出 CSV/Excel 格式

#### 3.2.5 视觉规范
- 卡片背景：白色，边框 1px #E2E8F0
- 圆角：16px，padding: 14px
- 标签字号：12px，颜色 #64748B
- 数值字号：14px，font-weight: 800，颜色 #605CE5
- 网格布局：2列，gap: 10px

---

### 3.3 关联网络区

#### 3.3.1 功能描述
可视化展示目标实体与其他实体（人员、组织、装备、技术）之间的关联关系。

#### 3.3.2 关系类型
| 关系类型 | 说明 | 示例 |
|----------|------|------|
| 研制关系 | 研发/生产方 | Kronshtadt → Orion |
| 使用关系 | 装备使用方 | 俄国防部 → Orion |
| 兼容关系 | 可挂载/配合使用 | Orion → Kh-39导弹 |
| 替代关系 | 新旧型号替代 | Orion → 改进型Orion |
| 竞争关系 | 同类产品竞争 | Orion vs 翼龙-2 |

#### 3.3.3 交互设计
- **图谱缩放**：支持鼠标滚轮缩放、拖拽平移
- **节点点击**：点击实体节点打开详情抽屉
- **关系筛选**：按关系类型筛选显示
- **路径追溯**：高亮显示两实体间的关联路径
- **布局切换**：力导向图/层次图/环形图

#### 3.3.4 视觉规范
- 节点大小：根据实体重要性动态计算
- 关系线：不同颜色区分关系类型
- 高亮效果：选中节点及关联节点高亮，其他淡化

## 4. 交互设计规范

### 4.1 实体详情抽屉

#### 4.1.1 触发方式
- 点击文章中的实体高亮文字
- 点击悬浮卡片
- 点击实体列表中的卡片
- 点击关联图谱中的节点

#### 4.1.2 抽屉规格
| 属性 | 规格 |
|------|------|
| 宽度 | 580px |
| 位置 | 右侧滑出 |
| 动画 | 300ms ease-out |
| 遮罩 | 黑色 50% 透明度 |

#### 4.1.3 内容布局
```
┌─────────────────────────────┐
│  图片轮播区                   │
├─────────────────────────────┤
│  标签行                       │
│  实体名称（原文 + 中文）        │
│  承包商 · xxx  别名 · xxx     │
├─────────────────────────────┤
│  简介                         │
├─────────────────────────────┤
│  核心战技指标（KPI卡片）       │
├─────────────────────────────┤
│  技术规格                     │
├─────────────────────────────┤
│  挂载装备                     │
├─────────────────────────────┤
│  主要使用方                   │
├─────────────────────────────┤
│  动态追踪（时间轴）            │
└─────────────────────────────┘
```

### 4.2 悬浮卡片

#### 4.2.1 触发方式
- hover 文章中的实体高亮文字
- 延迟 300ms 显示，避免频繁触发

#### 4.2.2 卡片内容
- 实体类型标签
- 实体名称（原文+中文）
- 核心简介（2-3行）
- 缩略图

#### 4.2.3 交互行为
- 点击卡片打开实体详情抽屉
- 点击卡片内的"查看详情"按钮
- 鼠标移出 500ms 后自动消失

### 4.3 文章内实体交互

#### 4.3.1 实体高亮
- 实体名称以特殊颜色/下划线标识
- hover 显示悬浮卡片
- 点击打开实体详情抽屉

#### 4.3.2 交互反馈
- hover：文字颜色变化 + 下划线
- 点击：高亮闪烁反馈
- 关联：右侧实体列表对应卡片高亮
